# Florence

## 概要
Florenceは実家の両親が経営している女子学生専用アパートです。
そのアパートを紹介するWebサイトです。
## 背景
現状はWebサイトが無く、入居者も年々減っている課題がありました。
Webからもアパートの情報がわかるようにし、より多くの入居希望者を獲得したいと考えたため、サイトを作成しようと思いました。
## ターゲット
群馬大学医学部に入学する予定の女子学生
10代〜20代
